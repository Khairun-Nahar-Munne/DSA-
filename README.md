# Codes of Data Structure and Algorithm

This repository contains implementations of various Data Structures and Algorithms (DSA) in C++, designed to help understand and solve computational problems efficiently.

## Contents

### Data Structures
- **Array**
- **Linked List**
  - Singly Linked List
  - Doubly Linked List
- **Stack**
- **Queue**
  - Circular Queue
  - Priority Queue
- **Tree**
  - Binary Tree
  - Binary Search Tree
  - AVL Tree
- **Graph**
  - Adjacency List
  - Adjacency Matrix
- **Hashing**
  - Hash Maps
  - Hash Sets

### Algorithms
- **Sorting**
  - Bubble Sort
  - Merge Sort
  - Quick Sort
  - Heap Sort
- **Searching**
  - Binary Search
  - Depth-First Search (DFS)
  - Breadth-First Search (BFS)
- **Dynamic Programming**
  - Longest Common Subsequence (LCS)
  - Knapsack Problem
- **Greedy Algorithms**
  - Activity Selection
  - Huffman Encoding
- **Graph Algorithms**
  - Dijkstra's Algorithm
  - Kruskal's Algorithm
  - Prim's Algorithm
- **Others**
  - Backtracking (e.g., N-Queens Problem)
  - Divide and Conquer
